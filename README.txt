== Description ==

WP vtiger plugin is a simple, advanced, easy to setup vtiger wordpress plugin as replacement of vtiger web form feature.

Note: WP Tiger is merged into WP Leads Builder for any CRM plugin along with wordpress vtiger, Zoho, Salesforce, Sugar CRM integration features. To upgrade for [Leads Builder For CRM Pro](https://www.smackcoders.com/wp-leads-builder-any-crm-pro.html) CRM Leads Generator now supports Vtiger CRM, Sales force, Zoho CRM and SugarCRM.

User can create, edit, update and manage unlimited web forms as short codes using wordpress vtiger crm lead capture plugin, which can be used to capture leads, contacts or accounts. No html code or visible Public ID in source like vtiger html web forms. Instead WP Tiger works with API integration and produce short codes which can be used anywhere in post, page, custom posts, even sidebar, widgets and popups. Added to this Captcha security available to avoid spamming. In addition, no mapping or Webform Reference Field matching required. Just directly pull out whole field set from selected Vtiger Module and configure their behavior, label etc., from an easy to setup user interface.

In total, it helps to generate a webforms via user-friendly crm wordpress leads plugin interface, which in turn used as a contact form to convert your traffic/visitors as high potential leads to VTiger CRM.  

- No HTML or visible public ids   
- Generated as dynamic shortcodes, used anywhere in post, page, custom posts, even sidebar, widgets and popups.
- No Manual Work, allows you to update multiple form elements by editing single shortcode  
- Simple Setup, pull fields from Vtiger CRM modules 
- User friendly UI, edit and update form elements from one place. 
- More Options like assign a form to one of the available Vtiger user 
- Wordpress vtiger crm lead capture plugin dynamically generate web forms directly from vtiger module fields
- Easy Manageable, plug and play model via API integration 


**Features**
 
* Generate vtiger wordpress form with or without web forms.
* Supports Contact Form 7 plugin.
* Use vtiger wordpress plugin forms in Page, Post, popups, optins or Widgets.
* Captures potential leads to Vtiger CRM, SugarCRM, Zoho CRM and Salesforce.
* Redirects User to the desired page after form submission.
* Notification on success or failure of lead capture.
* vtiger plugins convert new Users as your CRM contact.
* Vtiger wordpress ecommerce integration

== Installation ==

WP Leads Builder For CRM can be installed in usual 3 methods

I. For simple general way to install

    * Go to Plugins → Add New →Upload
    * Browse and Upload the downloaded zip file
    * Activate the plugin.

II. For familiar FTP users

    * Extract the plugin zip
    * Upload plugin zip to /wp-content/plugins/
    * Go to Plugins → Installed Plugins → Active

III. Straight from WordPress Admin

    * Go to plugins → Add New
    * Search for WP Leads Builder Any CRM
    * Click Install Now → Activate

== Frequently Asked Questions ==

1) How to create form with Contact Form 7?<br />

* Enable the Contact Form 7 in CRM configuration under Form Type.
* In CRM Forms menu select the required fields and Save.
* The shortcode with selected fields will be created in Contact Form 7.
* Place the shortcode in Post, Page or Widget.

2) Why I'm not able to submit the form created with Contact form?<br />

* After generating the shortcode for Contact form.
* Go to Contact >> Contact Forms >> Edit view of the shortcode.
* In Mail menu, configure all the settings and save.
* If the mail is not configured the form will trigger an error message stating 'Submission failed'.
